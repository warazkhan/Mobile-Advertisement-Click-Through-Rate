# Mobile Advertisement (CTR) - Machine Learning

The project involves step-by-step analysis of exploratory data (EDA), Data visualization, Feature engineering, Data Processing techniques, and finally Model selection. The dataset includes information about the click-through rate (CTR). It is a very critical metric for evaluating the performance of online ads.
